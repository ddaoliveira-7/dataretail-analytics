# 💻 SQL Analysis

## Overview

This folder contains SQL queries developed to analyze the retail sales database.

The queries were created to answer real business questions involving revenue, customers, products, salespeople, payment methods, cancellations, and sales targets.

---

## SQL Skills Demonstrated

- `SELECT`
- `WHERE`
- `ORDER BY`
- `LIMIT`
- Aggregate functions
- `GROUP BY`
- `HAVING`
- `INNER JOIN`
- Multiple table joins
- Calculated columns
- Business KPI analysis
- Revenue analysis
- Sales performance analysis

---

## Query Files

| File | Description |
|---|---|
| `01_basic_queries.sql` | Filtering, sorting, and basic data exploration |
| `02_joins.sql` | Queries connecting customers, orders, products, and salespeople |
| `03_business_analysis.sql` | Business questions and analytical queries |
| `04_kpis.sql` | Revenue, average ticket, targets, and performance indicators |

---

## Main Business Questions

- What are the most recent orders?
- What are the highest-value delivered orders?
- Which products have the highest prices?
- Which customer segment generates the most revenue?
- Which payment method has the highest cancellation rate?
- Which salespeople achieved their targets?
- What are the most valuable orders?
- Which products were included in a specific order?
