# 📈 Power BI Dashboard

## Overview

This folder contains the interactive Power BI dashboard developed for the Data Retail Analytics project.

The dashboard transforms transactional retail data into executive-level insights through KPIs, DAX measures, and interactive visualizations.

---

## Dashboard Features

- Executive KPI Cards
- Revenue Analysis
- Customer Segmentation
- Salesperson Performance
- Regional Analysis
- Product Performance
- Payment Method Analysis
- Interactive Filters

---

## KPIs

The dashboard includes:

- 💰 Total Revenue
- 📦 Delivered Orders
- 💳 Average Order Value
- ❌ Cancellation Rate
- 🏆 Top Salesperson
- ⭐ Best Selling Product

---

## Visualizations

- Revenue by Region
- Revenue by Customer Segment
- Monthly Revenue Trend
- Salesperson Ranking
- Top Products
- Payment Method Distribution

---

## DAX Measures

Examples of measures used:

- Total Revenue
- Average Ticket
- Delivered Orders
- Cancellation Rate
- Revenue by Region
- Revenue by Salesperson

---

## Data Model

The project uses a relational model connecting:

- Customers
- Orders
- Order Items
- Products
- Salespeople
- Targets
